(R) For the development of this project, I need to:
1. operate on vectors: to be consistent with a linear algebra definition of a vector, 
vectors only represent direction and length;
2. draw a line between them;
3. use colours to represent the line;
4. generate two exact copies of the same vector space;
5. graphically show two vector spaces in a single UI;
6. let user choose two transformations applied to all vectors in a vector space;
7. generate the same two transformations in reverse order;
8. apply the chosen transformations to the two vector spaces: one vector space transformed with the transformations in order AB, the other vector space transformed with the transformations in order BA;
9. draw the transformed vector space;
